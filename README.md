# GCD-by-Long-Division

Calculates Greatest Common Divisor using the Euclidean algorithm with remainder display.
Algorithm:
Divide larger number by smaller number
Replace larger with smaller, smaller with remainder
Repeat until remainder is 0
GCD is the last non-zero remainder
Example:
plain
Input: 12 32
Output:
remainder: 12
remainder: 8
remainder: 4
remainder: 0
The gcd of 12 and 32 is 4
